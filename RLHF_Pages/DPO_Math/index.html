    深度学习中的直接偏好优化（DPO）数学基础¶

    本文将系统地介绍直接偏好优化（Direct Preference Optimization, DPO）的数学基础。我们将从必要的数学前置知识入手，逐步展开DPO的数学推导，特别关注以下几个关键问题：

    1. DPO的梯度动力学
    2. 参数\(\beta\)的作用
    3. DPO中KL散度的包含方式

    前置知识¶

    Bradley-Terry模型基本概念¶

    Bradley-Terry模型是一种概率模型，用于描述配对比较（paired comparisons）中的偏好关系。其基本形式如下：

    对于任意两个选项\(i\)和\(j\)，选择\(i\)优于\(j\)的概率为：

    \[ P(i \succ j) = \frac{\pi_i}{\pi_i + \pi_j} \]

    其中： - \(\pi_i > 0\) 表示选项\(i\)的潜在“能力值”或“强度” - \(\succ\) 表示“优于”关系

    数学性质¶

    1. 对称性：

      \[ P(i \succ j) + P(j \succ i) = 1 \]

    2. 传递性：

      如果 \(P(i \succ j) > 0.5\) 且 \(P(j \succ k) > 0.5\)，则 \(P(i \succ k) > 0.5\)。

    KL散度基本概念¶

    KL散度（Kullback-Leibler散度）是一种用于衡量两个概率分布差异的度量。对于离散概率分布\(P\)和\(Q\)，KL散度定义如下：

    \[ D_{KL}(P \parallel Q) = \sum_{x} P(x) \log \frac{P(x)}{Q(x)} \]

    其中： - \(P(x)\)和\(Q(x)\)分别表示两个概率分布在事件\(x\)上的概率 - 对数通常采用自然对数（以\(e\)为底）

    数学性质¶

    1. 非负性：

      \[ D_{KL}(P \parallel Q) \geq 0 \]

      当且仅当\(P\)和\(Q\)完全相同时，KL散度为0。

    2. 不对称性：

      \[ D_{KL}(P \parallel Q) \neq D_{KL}(Q \parallel P) \]

      KL散度不是一个对称的度量。

    3. 凸性：

      KL散度关于其参数是凸函数。

    4. 加性：

      对于独立随机变量，联合分布的KL散度等于边缘分布KL散度之和。

    5. 连续性：

      对于连续概率分布，KL散度可以表示为积分形式：

      \[ D_{KL}(P \parallel Q) = \int P(x) \log \frac{P(x)}{Q(x)} \, dx \]

    DPO符号表¶

    符号 含义
    \(\pi_\theta(y \mid x)\) 策略模型（policy model）的条件概率分布
    \(\pi_\text{ref}(y \mid x)\) 参考模型（reference model）的条件概率分布
    \((x, y_w, y_l)\) 偏好数据三元组，包含上下文\(x\)，获胜回复\(y_w\)和失败回复\(y_l\)
    \(r_\theta(x,y)\) 奖励模型（reward model），定义为 \(\log(\pi_\theta(y \mid x)) - \log(\pi_\text{ref}(y \mid x))\)
    \(\beta\) 温度参数，控制策略的保守程度

    DPO Loss推导¶

    DPO的核心思想是将偏好学习转化为二分类问题。以下是DPO损失函数的逐步推导过程。

    1. 胜概率的表示¶

    根据Bradley-Terry模型，给定偏好对\((y_w, y_l)\)，选择\(y_w\)优于\(y_l\)的概率为：

    \[ P(y_w \succ y_l \mid x) = \frac{\exp(r_\theta(x, y_w))}{\exp(r_\theta(x, y_w)) + \exp(r_\theta(x, y_l))} \]

    其中，奖励模型定义为：

    \[ r_\theta(x, y) = \log(\pi_\theta(y \mid x)) - \log(\pi_\text{ref}(y \mid x)) \]

    2. DPO的损失函数¶

    DPO的损失函数旨在最大化获胜回复\(y_w\)被选中的概率。损失函数定义为负对数似然：

    \[ \mathcal{L}_\text{DPO}(\theta) = -\mathbb{E}_{(x, y_w, y_l)}\left[\log\left(\frac{\exp(r_\theta(x, y_w)/\beta)}{\exp(r_\theta(x, y_w)/\beta) + \exp(r_\theta(x, y_l)/\beta)}\right)\right] \]

    其中，引入了温度参数\(\beta\)，用于调节奖励的缩放。

    DPO Loss的数学性质¶

    1. 单调性¶

    • 当 \(\beta \to 0\) 时，损失函数趋向于硬分类损失，此时模型力图使得\(y_w\)的概率尽可能接近1，\(y_l\)接近0。
    • 当 \(\beta \to \infty\) 时，模型行为趋近于参考模型，即策略模型\(\pi_\theta\)与参考模型\(\pi_\text{ref}\)相似。

    2. 梯度性质¶

    我们将详细推导DPO损失函数的梯度表达式。

    DPO损失函数回顾¶

    \[ \mathcal{L}_\text{DPO}(\theta) = -\mathbb{E}_{(x, y_w, y_l)}\left[\log\left(\frac{\exp(r_\theta(x, y_w)/\beta)}{\exp(r_\theta(x, y_w)/\beta) + \exp(r_\theta(x, y_l)/\beta)}\right)\right] \]

    梯度推导步骤¶

    1. 简化表示

      令： - \(r_w = \frac{r_\theta(x, y_w)}{\beta} = \frac{\log(\pi_\theta(y_w \mid x)) - \log(\pi_\text{ref}(y_w \mid x))}{\beta}\) - \(r_l = \frac{r_\theta(x, y_l)}{\beta} = \frac{\log(\pi_\theta(y_l \mid x)) - \log(\pi_\text{ref}(y_l \mid x))}{\beta}\)

      则损失函数可以表示为：

      \[ \mathcal{L} = -\log\left(\frac{\exp(r_w)}{\exp(r_w) + \exp(r_l)}\right) \]

      进一步化简：

      \[ \mathcal{L} = -r_w + \log\left(\exp(r_w) + \exp(r_l)\right) \]

    2. 计算梯度

      对损失函数\(\mathcal{L}\)关于参数\(\theta\)求梯度：

      \[ \nabla_\theta \mathcal{L} = -\nabla_\theta r_w + \frac{\exp(r_w) \nabla_\theta r_w + \exp(r_l) \nabla_\theta r_l}{\exp(r_w) + \exp(r_l)} \]

    3. 引入概率变量

      令：

      \[ p = \frac{\exp(r_l)}{\exp(r_w) + \exp(r_l)} \\ 1 - p = \frac{\exp(r_w)}{\exp(r_w) + \exp(r_l)} \]

      则梯度表达式可以重写为：

      \[ \nabla_\theta \mathcal{L} = -(1 - p) \nabla_\theta r_w + p \nabla_\theta r_l \]

    4. 考虑\(\beta\)系数

      由于\(r_w\)和\(r_l\)中包含\(\beta\)，最終梯度表达式为：

      \[ \nabla_\theta \mathcal{L}_\text{DPO} = -\frac{1}{\beta} \mathbb{E}_{(x, y_w, y_l)}\left[(1 - p) \nabla_\theta r_\theta(x, y_w) - p \nabla_\theta r_\theta(x, y_l)\right] \]

    梯度动力学的解释¶

    • (1 - p) 与 \(y_w\)的梯度：表示获胜回复\(y_w\)的贡献。
    • p 与 \(y_l\)的梯度：表示失败回复\(y_l\)的贡献。

    通过这种组合，梯度更新同时考虑了奖励模型对获胜和失败回复的影响。

    DPO与KL散度的关系¶

    DPO在优化过程中隐式地控制了策略模型\(\pi_\theta\)与参考模型\(\pi_\text{ref}\)之间的KL散度。以下是详细的数学分析。

    1. 奖励模型与KL散度的关联¶

    回顾奖励模型的定义：

    \[ r_\theta(x, y) = \log \frac{\pi_\theta(y \mid x)}{\pi_\text{ref}(y \mid x)} \]

    这实际上可以表示为：

    \[ \pi_\theta(y \mid x) = \pi_\text{ref}(y \mid x) \exp(r_\theta(x, y)) \]

    2. KL散度的定义¶

    在DPO的上下文中，KL散度定义为：

    \[ D_{KL}(\pi_\theta \parallel \pi_\text{ref}) = \mathbb{E}_{y \sim \pi_\theta} \left[ \log \frac{\pi_\theta(y \mid x)}{\pi_\text{ref}(y \mid x)} \right] \]

    将奖励模型代入：

    \[ D_{KL}(\pi_\theta \parallel \pi_\text{ref}) = \mathbb{E}_{y \sim \pi_\theta} [r_\theta(x, y)] \]

    3. 优化过程中的KL散度控制¶

    DPO的优化目标隐式地包括了对KL散度的控制。具体体现在以下几个方面：

    优化目标函数¶

    DPO的目标函数为：

    \[ L(\theta) = \mathbb{E}_{(x, y_w, y_l)} \left[ \log \sigma \left( \beta \left( r_\theta(x, y_w) - r_\theta(x, y_l) \right) \right) \right] \]

    其中，\(\sigma\)为Sigmoid函数，\(\beta\)为温度参数。

    KL散度的约束¶

    根据KL散度的定义，DPO的优化过程中需要满足：

    \[ D_{KL}(\pi_\theta \parallel \pi_\text{ref}) = \mathbb{E}_{y \sim \pi_\theta} \left[ \log \frac{\pi_\theta(y \mid x)}{\pi_\text{ref}(y \mid x)} \right] \]

    在DPO中，通过奖励模型与策略模型的关系，KL散度被隐式地包含在优化过程中。

    温度参数\(\beta\)的作用¶

    温度参数\(\beta\)在调节KL散度的约束强度方面起着关键作用：

    • 较小的\(\beta\)：允许更大的KL散度，即策略模型可以更自由地偏离参考模型。
    • 较大的\(\beta\)：限制策略模型与参考模型之间的偏离，保持较小的KL散度。

    4. 优化过程分析¶

    DPO目标函数的展开¶

    将奖励模型展开，DPO的目标函数可以表示为：

    \[ L(\theta) = \mathbb{E}_{(x, y_w, y_l)} \left[ \log \sigma \left( \beta \left( \log \frac{\pi_\theta(y_w \mid x)}{\pi_\text{ref}(y_w \mid x)} - \log \frac{\pi_\theta(y_l \mid x)}{\pi_\text{ref}(y_l \mid x)} \right) \right) \right] \]

    KL散度的隐式控制¶

    在优化过程中，\(\beta\)参数通过以下机制控制KL散度：

    1. \(\beta\)乘以奖励差值：

      \[ \beta \left( \log \frac{\pi_\theta(y_w \mid x)}{\pi_\text{ref}(y_w \mid x)} - \log \frac{\pi_\theta(y_l \mid x)}{\pi_\text{ref}(y_l \mid x)} \right) \]

    2. \(\beta\)的影响：

      • 较大\(\beta\)：即使奖励差值较小，\(\beta\)乘以后会产生较大的梯度，导致优化过程对策略偏离参考模型更加敏感。因此，模型倾向于保持与参考模型较为接近的行为分布。
      • 较小\(\beta\)：需要较大的奖励差值才能产生显著梯度，优化过程对策略偏离参考模型较为宽松，允许更大的策略调整。

    5. 数学上的解释¶

    奖励模型中的比值 \(\log \frac{\pi_\theta(y \mid x)}{\pi_\text{ref}(y \mid x)}\) 正是KL散度的一部分。通过调节\(\beta\)，DPO在优化过程中有效地控制了这个比值的变化范围，从而间接地控制了KL散度。

    6. 直观理解¶

    • \(\beta\)作为“保守程度”调节器：
      • 大\(\beta\)：保守优化，保持小的KL散度，策略模型接近参考模型。
      • 小\(\beta\)：激进优化，允许较大的KL散度，策略模型可以更自由地偏离参考模型。

    这种设计使得DPO在最大化奖励的同时，通过\(\beta\)参数软约束KL散度，从而实现了策略模型与参考模型之间的平衡。
